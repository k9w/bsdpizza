We are a group of tech enthusiasts who use and enjoy Operating Systems descended from Berkeley Software Distribution (BSD).

We meet in or near Portland, Oregon, USA to discuss the topics below and others we find interesting.

  • OpenBSD
  • FreeBSD
  • NetBSD
  • DragonflyBSD
  • Linux
  • Free and Open Source Software

If you're in the area and want to join, search BSD Pizza Night on Calagator, a website for tech events in Portland.

For more information, contact:

  • Andrew Hewus Fresh Mastodon
  • Kevin Williams Mastodon Twitter

This site currently uses hand-written HTML and CSS and is hosted with an OpenBSD VM on a VMWare ESXi Dell server owned by one BSD Pizza member, racked at another member's house in the Portland area, and VPN'ed out to its public IP in Eugene (southern Oregon).
